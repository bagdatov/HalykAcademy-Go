Домашнее задание:

Необходимо реализовать функцию Unmarshal в плоскую структуру.
Функция, точно так же, как и функция из встроенного пакета Unmarshal
должна уметь принимать любую структуру.
Необходимо также перед началом распаковки JSON проверить, что аргумент
dst имеет тип "указатель на структуру". После этого нужно заполнить поля
структуры.
* Если поле JSON имеет стандартный тип (целое число, число с плавающей точкой, 
bool, null), то в случае наличия поля структуры с ТОЧНО таким же именем - поле
структуры заполняется.
* Если поле JSON является другим JSON объектом, то поле в go структуре должно иметь вид:

"$prefix" + "$field"
где $prefix - последовательность имен полей вложенных JSON объектов, а $field - имя поля
во вложенном JSON объекте

В файле main.go есть тестовая структура и JSON объект, который вы должен "распаковать" в
данную структуру.

Разрешается пользоваться стандартной библиотекой "encoding/json"
Удачи!:)

Задания повышенной сложности:
1. Сделать регистронезависимый поиск имен в полях структуры go
2. Добавить в структуру новые тэги следующего вида:
   `flatjson:"$name[/$path]"`
   где $name - имя JSON поля, которое должно ассоциироваться с данным полем структуры, а 
   $path - путь до какого-либо вложенного JSON поля
   Пример использования:
   Если у поля структуры есть тэг `flatjson:"Education/Faculty/Name"`, то с этим полем
   должно ассоциироваться вложенное поле из JSON структуры
3. Реализовать функцию Marshal
